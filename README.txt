All'interno della presente cartella viene riportato il materiale richiesto dalla
consegna del progetto di Applicazioni della Ricerca Operativa (Laurea Magistrale
in Informatica, A.A. 2020/21).

Nel dettaglio, all'interno della presente cartella è possibile trovare i
seguenti file/sottocartelle:
	- "relazione.pdf": la relazione testuale richiesta dalla consegna;
	- "presentazione.pdf": la presentazione richiesta dalla consegna;
	- "src/": cartella contenente gli script Python sui quali si basa
		l'implementazione proposta;
	- "data/": cartella contenente le istanze benchmark sulle quali è stata
		testata l'implementazione proposta;
	- "initial_solutions/": cartella che contiene, per ogni istanza, le
		soluzioni iniziali date dall'esecuzione dell'algoritmo VNS (utili ad
		ottimizzare le tempistiche qualora vengano eseguite più run su una
		singola istanza);
	- "results/": cartella contenente i risultati delle varie run eseguite sulle
		istanze benchmark (in formato .csv);
	- "results-comparison-table.xlsx": foglio Excel contenente un resoconto dei
		risultati ottenuti, comprensivi di confronti qualitativi effettuati
		con altri algoritmi presenti in letteratura (si noti che tali risultati
		sono comunque riportati anche nella relazione di progetto).

Qualora si volesse testare il funzionamento degli script proposti (ovvero quelli
contenuti all'interno del primo livello della cartella "src/"), si invita a
consultarne le rispettive guide all'utilizzo. Tali guide possono essere
consultate lanciando gli script nel seguente modo:

	$ python <nome_script> -h

Per ulteriori dettagli riguardanti l'implementazione proposta si rimanda ai
commenti contenuti all'interno dei diversi script.
